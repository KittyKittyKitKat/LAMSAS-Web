Welcome to the LAMSAS Query Web App

This is an application currently under development for the purpose of searching the vast amounts of data contained within the Linguistic Atlas Project (LAP). The LAP serves to catalogue the dialect of American English by region. More information on the project itself can be found at the project's website. Currently, this site uses data from a subset of the LAP, the Linguistic Atlas of the Middle and South Atlantic States (LAMSAS), to query and display information. This site is being developed by Simon Harris-Palmer under the mentorship of Dr. Jerzy Jaromczyk of the University of Kentucky as part of a Capstone Research Project. At the moment, this site runs on Python 3.9.1, using Flask for web application interfacing and SQLite for data storage and querying. Thanks to Dr. Allison P. Burkette for providing the opportunity to work with the LAP's data for the purposes of this project.
