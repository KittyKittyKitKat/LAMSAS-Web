LAMSAS Query Help

Simple Features

Data Table

This dropdown menu is used to select one of the two tables, Informants or Responses, currently within the database to draw data from. (Required)

Columns from Data Table

This selection dialogue is used to choose the columns you wish to retrieve based on the data table you chose. You can select more than one column to query by holding Control and clicking on multiple columns. (Required)

Order With

This dropdown menu is used to order results by the values in a column queried.

Distinct Results

Check this checkbox in order to return only unique data. This takes into account all the columns requested in the query.

Limit Results To

Limit the number of results returned by the query.

Advanced Features

Empty Values Last

When a column is specified in the Order With dropdown, this will make entires with no data appear last in the results.

Offset Results By

Offset the returned queries. This requires a limit to be specified. (For example, a limit of 10 would return results 1-10, whereas a limit of 10 and an offset of 2 would return results 3-12.)

Data Sorting Condition

These 3 fields are used to sort the data returned by the query. The leftmost field allows a column to be chosen from the previously selected columns. The center field contains conditions to test the values of the column with, and the rightmost allows you to enter in the specific value to test against. (For example, ColumnA | Is Equal to | Value would only return results where the value in ColumnA is equal to the specified Value.)

Display Query as SQLite

Display the query in SQLite syntax.
